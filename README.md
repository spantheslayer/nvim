# NeoVim Setup 
**Install NeoVim**

    sudo pacman -S neovim

**Clone the repo ⤵️**

    cd .config && git clone https://github.com/spantheslayer/nvim.git && cd nvim 
**Open `nvim`**
	

    :PlusInstall     (hit enter)



 **Nice. You are done :D**
